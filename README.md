# AI Package Delivery Optimization System
Web-based package delivery optimization system using AI algorithms


**Developed by:**
Duha Imad
Leen Anas

---

## Project Description

This project addresses the problem of efficiently assigning packages to delivery vehicles using Artificial Intelligence techniques. Two optimization approaches were implemented to improve the distribution process:

* Genetic Algorithm
* Simulated Annealing

A web-based interface was built using Flask, enabling users to interact with the system, upload input data, and view the generated results directly in the browser.

For a complete explanation of the implementation details and evaluation, please refer to the included PDF report.

## Running the Application

### Project Layout

Ensure the project files are organized as follows:

```
project/
├── app.py
├── templates/
│   └── (HTML files)
└── static/
    └── uploads/
```

---

### Steps to Run

1. Install the required package:

```
pip install flask
```

2. Start the server:

```
python app.py
```

3. Open your browser and navigate to:

```
http://127.0.0.1:5000
```

---

## Important Notes

* HTML pages must be placed inside the `templates` directory.
* The `uploads` folder inside `static` is used for handling uploaded files.
* Keep the main Python file (`app.py`) in the root folder of the project.
